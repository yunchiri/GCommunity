# GCommunity
Group Chat style Community backend

### 기능
- 그룹스타일
- Realtime chat
- 정렬, 기간 기능 강화
- 퍼오기 기능 업그레이드
- 댓글라시코 
- 모바일에 맞게 실시간성, 알람을 가장 신경씀

### DB tables

- Gtag (gid, target, id)
- Board
- Comment
- User

Group 관련 확장가능하도록
- player
- match
- team
- etc

OAuth 2.0 gwt token
redis
